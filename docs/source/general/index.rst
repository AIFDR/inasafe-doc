About |project_name|
====================

|project_name| is a plugin for QGIS software. It aims to produce realistic
natural hazard impact scenarios for better planning, preparedness and response
activities, using hazard and exposure geographic data.

Obviously you want to see how it looks like first. For that purpose we
prepared some nice :doc:`Screenshots <./screenshots>` for you.

Unluckily |project_name| has some :ref:`known_issues` which you should know
about before using the Software.
If you cannot find these issues happening anymore, please report back.
